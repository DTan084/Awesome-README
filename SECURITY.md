# Security Policy

## Supported Versions

Only the latest main branch of this project is actively supported and monitored for security vulnerabilities.

| Version | Supported |
| --- | --- |
| main (latest) | ✅ Yes |
| < 1.0.0 | ❌ No |

## Reporting a Vulnerability

If you discover a security vulnerability (such as an XSS vulnerability, HTML injection, or potentially malicious content inside templates), please do **NOT** open a public issue or pull request.

Instead, please report it privately:
1. Contact the maintainer Z0ZeroX on GitHub.
2. Email: **security@example.com** (placeholder — please update with your preferred security contact email)

We will review your report within 48 hours and work with you to patch the vulnerability and credit you for the discovery.
